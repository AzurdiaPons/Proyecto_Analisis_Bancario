Descripción del Proyecto
Este repositorio contiene una serie de consultas SQL desarrolladas para analizar el comportamiento, la lealtad y el perfil de riesgo de los clientes de una entidad bancaria. El objetivo principal es transformar datos brutos en insights accionables que permitan a la gerencia identificar segmentos de clientes en riesgo de abandono y optimizar las estrategias de retención.

Habilidades Técnicas Demostradas
En este análisis, apliqué las siguientes técnicas de SQL Server para la limpieza y exploración de datos:

Segmentación Avanzada: Uso de CASE WHEN para categorizar clientes según nivel de riesgo crediticio y rangos salariales.

Agregación y Análisis Estadístico: Implementación de funciones SUM, AVG y COUNT combinadas con CAST para obtener métricas exactas de deserción (Churn Rate).

Gestión de Calidad de Datos: Conversión de tipos de datos para permitir operaciones matemáticas precisas.

Análisis de Tendencias: Exploración de correlaciones entre la antigüedad del cliente (Tenure), número de productos contratados y comportamiento de abandono.

Reportes Estratégicos: Consultas optimizadas con GROUP BY y ORDER BY para priorizar segmentos VIP y realizar análisis geográficos de riesgo.

Estructura de los Scripts
01_Analisis_Riesgo_CreditScore.sql – Perfilamiento demográfico inicial.

02_Analisis_Lealtad_Productos.sql – Impacto de la diversificación de productos.

03_Segmentacion_VIP.sql – Identificación de clientes de alto valor en riesgo.

04_Analisis_Geografico.sql – Exposición de riesgo por país.

05_Analisis_Tenure.sql – Tasa de abandono según antigüedad del cliente.

06_Salario_vs_Riesgo.sql – Análisis de correlación económica.
